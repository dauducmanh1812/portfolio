# Đỗ Đức Mạnh — Portfolio

Personal portfolio built with **React + Vite + Tailwind CSS v4**, designed for instant Vercel deployment.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → Import your repo
3. Vercel auto-detects Vite — click **Deploy**
4. Every push to `main` auto-deploys

## Customize

- **Your info** → `src/data.js`
- **Profile photo** → replace `public/profile.jpg`
- **Theme color** → edit `--color-primary` in `src/index.css`

## Structure

```
src/
├── App.jsx, main.jsx, index.css, data.js
├── components/ (Navbar, Footer)
└── pages/ (HomePage, AboutPage, ProjectsPage)
```
